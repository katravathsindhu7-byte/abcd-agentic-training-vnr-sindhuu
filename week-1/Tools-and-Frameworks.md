# Agentic AI Tools and Frameworks

## Popular Agentic AI Frameworks

### 1. LangChain
- Agent and tool abstractions
- Memory support
- LLM orchestration

### 2. CrewAI
- Multi-agent collaboration
- Role-based agents
- Task delegation

### 3. AutoGPT
- Fully autonomous agents
- Long-running task execution

### 4. LangGraph
- Graph-based agent workflows
- State and memory handling

## Why These Tools Matter
These frameworks simplify agent creation by handling
planning, tool execution, and agent communication.
