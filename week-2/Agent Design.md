# Basic Agent Design

## Agent Components

1. Goal
- Defines what the agent wants to achieve

2. Planner
- Breaks the goal into smaller steps

3. Reasoning Engine
- Decides what action to take next

4. Action Executor
- Executes the selected action

5. Feedback Loop
- Evaluates progress and updates the plan

## Agent Type
- Single autonomous agent
- Rule-based planning
- Sequential execution

## Example Goal
"Generate a summary for a given topic"
